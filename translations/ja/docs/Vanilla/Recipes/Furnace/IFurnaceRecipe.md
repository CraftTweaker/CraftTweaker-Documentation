# IFurnaceRecipe

ICraftingRecipeは、ZSがそれを見る方法を炉テーブルのレシピです。

## クラスのインポート

エラーを避けるためにクラスを [インポート](/AdvancedFunctions/Import/) する必要があるかもしれません。  
`import crafttweaker.recipe.IFurnaceRecipe`

## ZenMethods/Getters

### To String

```zenscript
rec.commandString;
rec.toCommandString();
```

### その他の取得

```zenscript
rec.input;
rec.output;
rec.xp;
```

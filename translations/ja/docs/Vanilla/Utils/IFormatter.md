# IFormatter

## テキスト書式設定ハンドラへのアクセス

`` [global キーワード](/Vanilla/Global_Functions/) を使用して、書式設定ハンドラにアクセスできます。

## テキストの書式設定

### 色

以下のいずれかの色を使用して、IFormattedText に色を付けることができます。

- ブラック
- darkBlue
- ダークグリーン
- darkAqua
- darkRed
- ダークパープル
- ゴールド
- グレー
- darkGray
- 青
- 緑
- アクア
- 赤
- lightPurple
- 黄色
- 白

### スタイル

以下の書式スタイルの IFormattedText を提供することができます:

- 難読化
- bold
- 取り消し線
- underline
- italic

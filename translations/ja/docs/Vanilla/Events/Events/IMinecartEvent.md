# IMinecartEvent

このイベントは、Minecartsを含むすべてのイベントによって拡張されます。

## クラスのインポート
エラーを避けるためにクラスを [インポート](/AdvancedFunctions/Import/) する必要があるかもしれません。  
`import crafttweaker.event.IMinecartEvent;`

## 何が彼らを使って行うことができますか?

| 名前       | タイプ                                   |
| -------- | ------------------------------------- |
| minecart | [IEntity](/Vanilla/Entities/IEntity/) |

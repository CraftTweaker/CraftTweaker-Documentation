# Health Processor

## パッケージ
```zenscript
import mods.calculator.health;
```

## メソッド

- **[IIngredient](/Vanilla/Variable_Types/IIngredient/) input** 入力。
- **Int値** 入力の量がそのマシンに追加されます。

## Additon
```zenscript
health.addRecipe(IIngredient input, int value);
```

## 削除
```zenscript
health.removeRecipe(IIngredient入力);
```

# スターク抽出器

## パッケージ
```zenscript
import mods.calculator.starch;
```

## メソッド

- **[IIngredient](/Vanilla/Variable_Types/IIngredient/) input** 入力。
- **int value** 入力の量がそのマシンに追加されます。


## Additon
```zenscript
starch.addRecipe(IIngredient input, int value);
```

## 削除
```zenscript
starch.removeRecipe(IIngredient入力);
```

# レッドストーン抽出

## パッケージ
```zenscript
import mods.calculator.redstone;
```

## メソッド

- **[IIngredient](/Vanilla/Variable_Types/IIngredient/) input** 入力。
- **int value** 入力の量がそのマシンに追加されます。


## Additon
```zenscript
redstone.addRecipe(IIngredient input, int value);
```

## 削除
```zenscript
redstone.removeRecipe(IIngredient入力);
```

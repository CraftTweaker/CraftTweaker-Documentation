# IItemColorSupplier

IItemColorSupplier 関数は、 [Color Object](/Mods/ContentTweaker/Vanilla/Types/Color/Color/) を提供するために使用できます。

## パッケージのインポート

問題が発生した場合は、パッケージをインポートする必要があるかもしれませんので、申し訳ありませんし、インポートを追加してください。  
`import mods.contenttweaker.IItemColorSupplier;`

## パラメータ

IItemColorサプライヤー関数は、以下のパラメータを持つ関数です。

- [IItemStack](/Vanilla/Items/IItemStack/) itemStack → 要求されている色のアイテム。
- int tintIndex → 色素インデックスを使用します。 関数は [Color Object](/Mods/ContentTweaker/Vanilla/Types/Color/Color/) を返す必要があります。

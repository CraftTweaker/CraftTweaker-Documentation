# 工業的なフォアgoing

工業フォアゴーイングのバイオリアクター、レーザードリル、スラッジリファイナークラフト微調整サポート。

### 商品の重量

`ItemWeight` は、アイテムが選択される確率を表す数値です。 例:

| 項目 | 重量 | 確率 % |
| -- | -- | ---- |
| 石  | 99 | 99%  |
| 土  | 1  | 1%   |

| 項目 | 重量 | 確率 % |
| -- | -- | ---- |
| 石  | 2  | 66%  |
| 土  | 1  | 33%  |

ItemChance% = (ItemWeight/TotalWeight)*100

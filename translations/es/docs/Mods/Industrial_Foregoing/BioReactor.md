# Bioreactor

### Importando

```zenscript
import mods.industrialforegoing.BioReactor;
```

### Agregando

```zenscript
BioReactor.add(IItemStack input);

```

### Eliminando

```zenscript
BioReactor.remove(IItemStack input);

BioReactor.remove(<minecraft:diamond>);
```

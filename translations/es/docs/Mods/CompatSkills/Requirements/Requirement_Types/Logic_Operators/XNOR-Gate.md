# Operador XNOR

Una compuerta **XNOR** es una compuerta **XOR** invertida. Esto es lo mismo que usar la salida de la compuerta **XOR** que la entrada a la compuerta **NOT**. Esto significa que la compuerta **XNOR** devuelve `TRUE` s√≥lo cuando ambas entradas son el mismo estado. Ambos son `TRUE` o ambos son `FALSE`.

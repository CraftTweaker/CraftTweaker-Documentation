# Obsługa nawiasów

## Obsługa nawiasów

### Uchwyt z nawiasem umiejętności:

    ## Przykład:
    <skill:resourcelocation>
    
    ## Przykład:
    <skill:reskillable:agility>
    

### Obsługa nawiasów cech:

    ## Przykład:
    <trait:resourcelocation>
    
    ## Przykład:
    <trait:reskillable:sidestep>

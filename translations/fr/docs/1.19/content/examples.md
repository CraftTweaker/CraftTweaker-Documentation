# Examples

Vous pouvez utiliser la commande `/ct exemples` pour obtenir des scripts commentés dans votre répertoire scripts/exemples

Plus d'exemples de scripts peuvent être trouvés ici:

[Examples](https://github.com/CraftTweaker/CraftTweaker/tree/1.19/Common/src/main/resources/data/crafttweaker/scripts)

Keep in mind that examples are, after all, scripts, and that the next time you reload the game, they will be loaded. Examples may include undesired changes or functionality, so be careful with that.

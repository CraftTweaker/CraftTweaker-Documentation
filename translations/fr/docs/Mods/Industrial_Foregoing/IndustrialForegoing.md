### Poids de l'élément

Le `ItemWeight` est un nombre qui représente la chance d'un élément d'être sélectionné. Par exemple :

| Élément | Poids | Chance % |
| ------- | ----- | -------- |
| Pierre  | 99    | 99%      |
| Terre   | 1     | 1%       |

| Élément | Poids | Chance % |
| ------- | ----- | -------- |
| Pierre  | 2     | 66%      |
| Terre   | 1     | 33%      |

format@@0 ItemChance% = (ItemWeight/TotalWeight)*100

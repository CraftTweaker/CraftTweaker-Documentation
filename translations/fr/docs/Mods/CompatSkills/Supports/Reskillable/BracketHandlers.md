# Gestionnaires de tranches

## Gestionnaires de tranches

### Gestionnaire de la tranche de compétences :

    ## Exemple :
    <skill:resourcelocation>
    
    ## Exemple de Travail :
    <skill:reskillable:agility>
    

### Gestionnaire de tranchées :

    <trait:resourcelocation>
    
    ## Exemple de Travail :
    <trait:reskillable:sidestep>

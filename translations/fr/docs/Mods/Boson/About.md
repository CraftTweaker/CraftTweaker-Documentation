# À propos de Boson

Boson est un mod de bibliothèque en son cœur, ce qui signifie qu'il ne fournit pas de nouvelles fonctionnalités de jeu ou de mécanismes spéciaux, mais agit plutôt comme l'épine dorsale de certains autres mods, alléger le fardeau et simplifier certaines tâches répétitives ou difficiles à maîtriser durant le processus d'écriture.

Néanmoins, Boson contient quelques fonctionnalités supplémentaires qui peuvent intéresser le fabricant de pack ou un utilisateur plus général, comme info-bulles détaillées qui peuvent afficher les données NBT ou les entrées de dictionnaire de minerai. Reportez-vous à la liste des fonctionnalités plus tard dans cette page pour trouver lesquelles de ces fonctionnalités sont compatibles avec CraftTweaker.

Moreover, Boson also provides some classes that are part of the (unofficial) [ZenScriptX Project](/Mods/Boson/ZenScriptX/), which are a set of tools that greatly expand the potential of the ZenScript language. Reportez-vous à la page liée pour plus d'informations.

Avant de se plonger dans les fonctionnalités, il est fortement suggéré de lire la page [Noms dans Boson](/Mods/Boson/Names/) , pour quelques détails sur la façon dont les noms sont traités dans les interactions CraftTweaker de ce mod.

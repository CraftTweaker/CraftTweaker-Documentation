# IBiomeType


## 导入相关包
如果你遇到问题（例如创建[数组](/AdvancedFunctions/Arrays_and_Loops/)），你可能需要导入相关包，为此，最安全的方法就是进行导入。  
`import crafttweaker.world.IBiomeType;`

## ZenGetters

| ZenGetter | 返回值类型                                 |
| --------- | ------------------------------------- |
| name（名称）  | string                                |
| biomes    | 列表<[IBiome](/Vanilla/Biomes/IBiome/)> |

# Lubricant Registration

## 注册

| 类型 | 数据类型       |
| -- | ---------- |

### 序言

### Code Example:
```zenscript
mods.immersivepetroleum.Lubricant.register(IFluidStack 润滑油入口)；
mods.immersivepetroleum.Lubricant.register(<fluid:lava> * 500);
```

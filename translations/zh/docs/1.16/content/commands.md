# 命令

重要的指令：
```plaintext
/ct hand
#查看自己手上所拿着物品的id及其nbt标签(如果有的话)
```
```plaintext
/ct log
#查看mod所生成的日志
```
```plaintext
/ct scripts
#打开游戏目录下的scripts文件夹
```
```plaintext
/ct issues
#打开github链接向开发者提供问题反馈
```
```plaintext
/ct discord
#打开CT的官方discord频道
```
```plaintext
/ct dump
#在log文件中导出所需的所有信息
```
```plaintext
/ct dump recipeTypes
```
```plaintext
/ct dump_brackets
```
```plaintext
/ct help
#显示帮助信息
```
```plaintext
/ct recipes
```

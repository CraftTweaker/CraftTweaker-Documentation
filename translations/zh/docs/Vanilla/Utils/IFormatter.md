# IFormatter

## 访问文本格式化处理程序

您可以使用 `格式` [全局关键字](/Vanilla/Global_Functions/) 访问格式化处理器。

## 正在格式化文本

### 颜色

您可以使用以下颜色之一给一个 IFormatedText 颜色：

- 黑色
- darkBlue
- 深青色
- 深色水管
- 深红色
- 深紫色
- 金币
- 灰度
- 深灰度
- 蓝色
- 绿色的
- 水管
- 红色的
- 浅紫色
- 黄色
- 白色的

### 样式

您可以给以下格式化样式提供一个 IFormatText (或更多)：

- 迷惑的
- bold
- 删除线
- 下划线
- italic

# IData

IData接口是处理NBT等数据的通用接口。  
您可以投射所有原始文件 (短，双倍，字符串，...) 以及某些数组到 IData。  
记住, 当他们提供类似的功能时, IData和它们的对应机构并不一样，这就是为什么它们会被称为DataTypes (e)。 . DataBool。

## 导入相关包

如果您遇到任何问题，可能需要导入软件包(例如铸造一个 [数组](/AdvancedFunctions/Arrays_and_Loops/)), 这样比抱歉更安全并添加导入。  
`导入craftmilever.data;`

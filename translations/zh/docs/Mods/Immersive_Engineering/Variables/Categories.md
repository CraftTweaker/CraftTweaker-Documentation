# 类别管理

## 信息

蓝图有一个所需的字符串参数，叫做“类别”。 此字符串允许您将配方“分类”到特定的蓝图类别。

如果此字符串返回一个尚未存在的类别，它将创建该类别和一个适当的蓝图。 如果它找到一个现有的类别，它会将该配方添加到上述类别。

### 现有类别

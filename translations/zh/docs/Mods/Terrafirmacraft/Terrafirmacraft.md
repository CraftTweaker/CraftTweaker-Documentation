# Terrafiramacraft

## 信息
Terrafirmacraft The Next Generation(or TFC TNG for short) is a panilla remult model. 过渡联邦委员会的目标是使人们更加相信事情，让玩家有一种成就感。 掌握技能，并在远处广泛搜集材料，然后你才能建造惊人的城堡、房屋、堡垒、城镇等。 经过整修的地形生成还创造了一个美丽、多样的背景，作为开始你的建筑的画布。

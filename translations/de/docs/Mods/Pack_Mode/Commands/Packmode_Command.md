# Packmode-Befehl

## Anruf

Sie rufen das Kommando mit `/packmode newMode` auf, wobei `newMode` der Name des neuen PackMode ist

## Auslastung

Du verwendest den Befehl, um den Packmodus deines Spiels festzulegen.

# Обработчики скобок

## Обработчики скобок

### Обработчик Уровня Умений:

    ## Пример:
    <skill:resourcelocation>
    
    ## Рабочий пример:
    <skill:reskillable:agility>
    

### Обработчик скобок симптома:

    ## Пример:
    <trait:resourcelocation>
    
    ## Рабочий пример:
    <trait:reskillable:sidestep>

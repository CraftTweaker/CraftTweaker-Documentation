# Decay Generator

## Импортирование пакета
`mods.nuclearcraft.Генератор деканов`

## Добавление рецептов
```zenscript
mods.nuclearcraft.DecayGenerator.addRecipe(IIngredient blockInput, IIngredient blockOutput, двойное средство, двойная мощность, двойное излучение);
```

## Удаление рецептов
```zenscript
mods.nuclearcraft.DecayGenerator.removeRecipeWithInput(IIngredient blockInput);
mods.nuclearcraft.DecayGenerator.removeRecipeWithOutput(IIngredient blockOutput);
mods.nuclearcraft.DecayGenerator.removeAllRecipes();
```

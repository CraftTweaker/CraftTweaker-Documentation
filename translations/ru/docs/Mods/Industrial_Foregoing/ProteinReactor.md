# Белковый реактор

### Импорт

```zenscript
import mods.industrialforegoing.ProteinReactor;
```

### Добавление

```zenscript
ProteinReactor.add(IItemStack);

ProteinReactor.add(<minecraft:diamond>);
```

### Удаление

```zenscript
ProteinReactor.remove(IItemStack input);

ProteinReactor.remove(<minecraft:diamond>);
```

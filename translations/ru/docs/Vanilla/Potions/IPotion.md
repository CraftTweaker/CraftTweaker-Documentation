# Зелье

IPotion object ссылается на зелье в игре.

## Импорт пакета

Возможно, вам потребуется импортировать пакет, если вы столкнетесь с какими-либо проблемами (например, наложение [массива](/AdvancedFunctions/Arrays_and_Loops/)), так что лучше быть безопасным, чем извините и добавить импорт.  
`импортируйте crafttweaker.potions.IPotion;`

## Получение объекта IPotion

Такой объект можно получить с помощью [обработчика зелий](/Vanilla/Brackets/Bracket_Potion/)

```zenscript
<potion:minecraft:strength>;
```

## Геттеры

В настоящее время все, что вы можете сделать с зельями, получает некоторую информацию о них.  
Пример `<potion:minecraft:strength>.name`

| Геттер               | Что он делает                                       | Возвращаемый тип                                 |
| -------------------- | --------------------------------------------------- | ------------------------------------------------ |
| имя                  | Возвращает внутреннее имя зелья                     | string                                           |
| badEffect            | Возвращает если зелье плохо                         | bool                                             |
| ликвидационный цвет  | Возвращает цвет зелья                               | int                                              |
| жидкий цвет          | Возвращает цвет зелья                               | int                                              |
| целительные элементы | Возвращает список всех лечебных предметов для зелья | Список<[IItemStack](/Vanilla/Items/IItemStack/)> |
| имеет значок статуса | Возвращает если у зелья есть значок статуса         | bool                                             |
| Бенефициар           | Возвращает, выгодно ли зелье                        | bool                                             |
| Мгновенный           | Возвращает ли зелье мгновенно                       | bool                                             |

## ZenMethods

### Сделать эффектом зелья

[IPotionEffect](/Vanilla/Potions/IPotionEffect/) - это зелье, которое имеет длительность и усилитель. Вы можете использовать эти два метода, чтобы получить один из IPotion:  
Как можно угадать, оба возвращают объект [IPotionEffect](/Vanilla/Potions/IPotionEffect/).

```zenscript
pot.makePotionEffect(int duration, int amplifier);
pot.makePotionEffect(int duration, int amplifier, boolean ambientEffect, boolean particlesShown);
```

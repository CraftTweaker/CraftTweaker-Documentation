# Преобразователи предметов

Трансформаторы предмета преобразуют вводную информацию при создании предмета.  
Дальность действия может варьироваться от повреждения предмета до возврата совершенно другого предмета.

## Импорт пакета

Возможно, вам потребуется импортировать пакет, если вы столкнетесь с какими-либо проблемами (например, наложение [массива](/AdvancedFunctions/Arrays_and_Loops/)), так что лучше быть безопасным, чем извините и добавить импорт.  
`импорт crafttweaker.item.IItemTransformer;`

```zenscript
<br />val item = <minecraft:apple>;

//Предмет не будет потрачен и останется в сетке
преобразованный предмет = предмет. euse();

//Предмет не будет потрачен, и весь стек будет возвращен вам (дает).
transedItem = item.giveBack();

//предмет будет потребляться, но даст вам определенный стек (слот создания будет очищен!).
преобразованный элемент = элемент. iveBack(<minecraft:potato>);

//предмет будет заменен указанным предметом, который вместо этого перейдет в слот создания
преобразованный предмет = предмет. ransformReplace(<minecraft:potato>);

//повреждает элемент на 1
преобразованный элемент = элемент. ransformDamage();

//повреждает предмет заданным значением
transformedItem = item.transformDamage(3);

//предмет будет потребляться, вне зависимости от того, что.
transedItem = item.noReturn();

//Причина потребления нескольких предметов.
transformedItem = item.transformConsume(3);
```

## Регистрация собственных преобразователей предметов

### Преобразовать

Старый элемент трансформатора, который может перестать существовать в 1.13.  
Это специальная функция, которая принимает два параметра: предмет [](/Vanilla/Items/IItemStack/) сам и игрок [](/Vanilla/Players/IPlayer/) выполняющий Крафт.

```zenscript
transformedItem = item.transform(function(item, player) {return item;});
```

Функция должна вернуть [IItemStack](/Vanilla/Items/IItemStack/).  
Этот стек заменит то, что находится в слоте создания. Используйте `null` , чтобы очистить этот слот.

### Преобразовать Новый

С новой внутренней системой рецептов возникла необходимость в новом преобразователе ItemTransformer. Этот параметр принимает только один параметр, то есть элемент в слоте.

```zenscript
transformedItem = item.transformNew(function(item){return item;});
```

Функция должна возвращать [IItemStack](/Vanilla/Items/IItemStack/).  
Однако, в отличие от другого трансформатора это не будет предметом, заменяющим его в слоте для крафта, но тот, который возвращается для слота крафта.  
Другими словами, если вы вернете `нулевое` здесь, один элемент будет потребляться, любой возвращенный предмет будет помещен в слот для крафта, если это возможно, или возвращается вам, так же, как при работе с ведрами.  
Если вам не нужна переменная игрока, это трансформатор для этого!

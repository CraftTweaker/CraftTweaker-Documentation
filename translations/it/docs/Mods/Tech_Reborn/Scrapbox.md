# Scrapbox

## Importazione del pacchetto
`mods.techreborn.scrapbox`

## Aggiungere Gocce
```zenscript
mods.techreborn.scrapbox.addScrapboxDrop(IIngredient input);
```

## Rimozione Di Gocce
```zenscript
mods.techreborn.scrapbox.removeRecipe(IItemStack output);
mods.techreborn.scrapbox.removeAll();
```

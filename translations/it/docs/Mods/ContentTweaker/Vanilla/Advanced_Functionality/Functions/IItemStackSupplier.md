# IItemStackSupplier

La funzione IItemStackSupplier può essere aggiunta a una [scheda creativa](/Mods/ContentTweaker/Vanilla/Creatable_Content/Creative_Tab/) ed è utilizzata per fornire il [IItemStack](/Vanilla/Items/IItemStack/) da usare come icona della scheda.

## Importazione del pacchetto

Potrebbe essere necessario per importare il pacchetto se si verificano problemi, in modo da essere più sicuri di spiacenti e aggiungere l'importazione.  
`import mods.contenttweaker.IItemStackSupplier;`

## Parametri

La funzione IItemStackSupplier è una funzione senza parametri. La funzione deve restituire un [IItemStack](/Vanilla/Items/IItemStack/).

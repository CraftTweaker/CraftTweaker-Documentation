# Costruzione Del Tinker

## Avviso

A partire da CompatSkills 1.9.0, TConstruct Locks ora passa "su" a item-level. In questo modo gli strumenti sono ancora inaccessibili se un amico fornisce una versione artigianale.

## Comandi

    // Dump all Materials
    ~ /ct tinkermaterials
    
    // Dump all Modifiers
    ~ /ct tinkermodifiers

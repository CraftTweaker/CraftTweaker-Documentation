# Менеджеры рецептов


Все связанные с глобальными рецептами происходят из этого класса и имеют все эти методы.

```zenscript
removeRecipe(IItemStack output);
```

```zenscript
removeByName(String name);
```

```zenscript
removeByModid(String modid);
```

```zenscript
removeByRegex(String regex);
```

```zenscript
removeAll();
```

# Tarif Yöneticileri


Tariflerle ilgili bütün globaller bu sınıftan türemiştir ve aşağıdaki fonksiyonlara sahiptir.

```zenscript
removeRecipe(IItemStack output);
```

```zenscript
removeByName(String name);
```

```zenscript
removeByModid(String modid);
```

```zenscript
removeByRegex(String regex);
```

```zenscript
removeAll();
```

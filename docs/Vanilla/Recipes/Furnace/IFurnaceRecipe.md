# 烧炼配方
ICraftingRecipe是Zenscript所看到的烧炼配方。


## 导入相关包
可能需要[导入](/AdvancedFunctions/Import)相关包以避免错误。
`import crafttweaker.recipes.IFurnaceRecipe`


## ZenMethods/Getters
### To String
```
rec.commandString;
rec.toCommandString();
```

### Other getters
```
rec.input;
rec.output;
rec.xp;
```

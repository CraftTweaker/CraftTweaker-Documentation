# Facing

The Facing Interface allows you to get an [IBlockPos's](IBlockPos) possible direction Strings.

## Importing the package
It might be required for you to import the package if you encounter any issues, so better be safe than sorry and add the import.  
`import crafttweaker.world.Facing;` 

## Enumerations
The facing Interface has 6 static Methods, each returning the String of the said direction.
```
crafttweaker.world.Facing.north()
crafttweaker.world.Facing.east()
crafttweaker.world.Facing.south()
crafttweaker.world.Facing.west()
crafttweaker.world.Facing.down()
crafttweaker.world.Facing.up()
```

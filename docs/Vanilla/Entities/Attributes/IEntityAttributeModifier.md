# 实体属性修饰器

## 导入相关包
为了避免发生一些不期而遇的问题（比如声明[数组](/AdvancedFunctions/Arrays_and_Loops)），最为安全、也是最为推荐的方式就是导入相关的包。
`import crafttweaker.entity.AttributeModifier;`


## ZenGetters

| ZenGetter |  GetterMethod  | 返回类型 |
| :-------: | :------------: | :------: |
|   uuid    |   getUUID()    |  字符串  |
|   name    |   getName()    |  字符串  |
| operation | getOperation() |   整数   |
|  amount   |  getAmount()   | 双精度型 |

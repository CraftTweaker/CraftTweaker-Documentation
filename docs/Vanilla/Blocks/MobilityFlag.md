# MobilityFlag

The Facing Interface allows you to get an [IBlockState's](IBlockState) possible mobilityFlas Strings.

## Importing the package
It might be required for you to import the package if you encounter any issues, so better be safe than sorry and add the import.  
`import crafttweaker.block.MobilityFlag;` 

## Enumerations
The MobilityFlag Interface has 5 static Methods, each returning the String of the said direction.
```
crafttweaker.block.MobilityFlag.normal()
crafttweaker.block.MobilityFlag.destroy()
crafttweaker.block.MobilityFlag.block()
crafttweaker.block.MobilityFlag.ignore()
crafttweaker.block.MobilityFlag.pushOnly()
```

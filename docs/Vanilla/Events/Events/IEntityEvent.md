# 实体事件（IEntityEvent）

这个接口被所有拥有有生命实体的事件所继承。

## 导入相关类
最好在使用前 [导入](/AdvancedFunctions/Import) 类来避免错误。   
`import crafttweaker.event.IEntityEvent;`

# 方块材料


`方块材料` 使你可以定义方块在游戏中的质地
在Minecraft中,方块材料只有这些:

在定义方块材料时,方块材料应以尖括号引用的方式调用,如:

```
<blockmaterial:name>

<blockmaterial:wood>
```

如果你引用的方块材料是可用的,这将返回 [自定义材料](/Mods/ContentTweaker/Vanilla/Types/Block/IMaterialDefinition) 对象

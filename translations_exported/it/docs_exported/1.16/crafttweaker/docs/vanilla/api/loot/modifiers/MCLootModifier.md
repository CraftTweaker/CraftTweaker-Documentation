# MCLootModifier

Represents a vanilla or mod-added loot modifier.

## Importare la Classe

It might be required for you to import the package if you encounter any issues (like casting an Array), so better be safe than sorry and add the import at the very top of the file.
```zenscript
import crafttweaker.api.loot.modifiers.MCLootModifier;
```


## Interfacce Implementate
MCLootModifier implements the following interfaces. That means all methods defined in these interfaces are also available in MCLootModifier

- [ILootModifier](/vanilla/api/loot/modifiers/ILootModifier)

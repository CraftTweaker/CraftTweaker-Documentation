# Wearable

Used to denote armor items that can be worn.

## Importare la Classe

It might be required for you to import the package if you encounter any issues (like casting an Array), so better be safe than sorry and add the import at the very top of the file.
```zenscript
import crafttweaker.api.item.Wearable;
```


## Interfacce Implementate
Wearable implements the following interfaces. That means all methods defined in these interfaces are also available in Wearable

- [Vanishable](/vanilla/api/item/Vanishable)

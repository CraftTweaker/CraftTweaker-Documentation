# CraftTweaker-Documentation
Documentation for CraftTweaker

This is the source code for the documentation, you can find the actual site and documentation [here](http://docs.blamejared.com)


### Setup:

```
git submodule init 
git submodule update
npm install
cd site/
npm install
```

### Build

```
npm run build
```

### View Results

```
cd site
npm run dev
```
